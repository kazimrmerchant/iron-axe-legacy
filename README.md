# Iron Axe: Legacy

An original side-scrolling beat 'em up inspired by the classic Golden Axe formula.

**Completely original characters, story, and assets.**  
No Sega / Golden Axe copyrighted material is used.

## Vision
- Multiple playable fighters with unique movesets and magic
- Side-scrolling stages with elevation and multiple paths
- Combo system, throws, specials, and team magic
- Modern juice (hitstop, screen shake, particles)
- Built in **Godot 4**

## Project Status
This repository contains the full project foundation and core systems.
We will expand it system-by-system into a complete game.

## How to Run
1. Download / clone this repository
2. Open the project in **Godot 4.3+**
3. Run the main scene

## Characters (Planned)
- **Ragnar** – Heavy warrior (slow, high damage, strong grabs)
- **Lyra** – Agile fighter (fast combos, aerial focus)
- **Vex** – Magic-focused (weaker normals, strong spells)
